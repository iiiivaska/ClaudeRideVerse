## Glass-эффект

Liquid Glass используется только для floating UI -- tab bar, pills, HUD, кнопки поверх карты.

```
background: rgba(22,22,24,0.86)       // T.glass
backdrop-filter: blur(24px)
border: 1px solid rgba(255,255,255,0.10)  // T.glassB
```

Для tab bar усиленная версия:
```
background: rgba(20,20,22,0.90)
backdrop-filter: blur(32px) saturate(180%)
border: 0.5px solid rgba(255,255,255,0.13)
box-shadow: 0 12px 48px rgba(0,0,0,0.6),
            0 2px 8px rgba(0,0,0,0.4),
            inset 0 1px 0 rgba(255,255,255,0.07)
```

## Шрифты

### DM Sans (без засечек)
- Заголовки: 26-28px, weight 700
- Body: 13-16px, weight 400-600
- Кнопки: 14-16px, weight 600
- Используется для всего текстового контента, кроме чисел и лейблов

### IBM Plex Mono (моноширинный)
- Крупные метрики: 20-24px, weight 600, `line-height: 1`
- Средние значения: 11-14px, weight 500-600
- Лейблы (caps): 8-10px, weight 500, `letter-spacing: 0.12em`, `text-transform: uppercase`, цвет `tq`
- Используется для **всех чисел** без исключений, а также для меток секций и тегов

### Правило: числа всегда IBM Plex Mono
Любые числовые значения в интерфейсе (скорость, расстояние, hex count, время, elevation) набираются моноширинным шрифтом. Единицы измерения рядом с числом тоже моноширинные.

## Hex-ячейки

Все hex-ячейки -- pointy-top ориентация.

| Состояние | Fill | Stroke | Эффект |
|-----------|------|--------|--------|
| Fog (неисследованная) | `#3A3A3C` (bg3) | -- | -- |
| Visited-faded | `accent 40% opacity 0.5` | `accent 77` opacity | -- |
| Explored (открытая) | `accent 40%` (`#0A84FF38`) | `accent` 0.6px | -- |
| New unlock (новая) | `accent 100%` (`#0A84FF88`) | `accent` 1px | Glow: stroke 6px `accent 25%` opacity |

### Анимации hex
- `hexGlow`: пульсация opacity 0.6 -- 1.0, длительность ~1.8s, ease-in-out, бесконечная
- Unlock pulse: при открытии нового hex -- вспышка с accent glow

## Навигация

4 таба в floating glass tab bar:

| Таб | Иконка | Label |
|-----|--------|-------|
| Map | Home (стилизованный) | Map |
| Rides | Activity (кардиограмма) | Rides |
| Stats | Signal (radar) | Stats |
| Profile | Person | Profile |

Активный таб: accent цвет иконки и текста, `accS` фон, weight 600.
Неактивный: `rgba(235,235,245,0.4)`, weight 400.

## Анимации

| Имя | Описание |
|-----|----------|
| `rPulse` | Красный пульс REC-индикатора: box-shadow 0-7px, `rgba(255,69,58,0.6)` |
| `locPulse` | Пульс user location: r 10-22, opacity 0.4-0 |
| `hexGlow` | Мерцание hex: opacity 0.6-1.0 |
| `slideUp` | Появление sheet снизу: translateY(100%) -> 0 |

## Общие правила

1. **Числа** -- всегда IBM Plex Mono, без исключений
2. **Glass** -- только для floating UI поверх карты (tab bar, pills, HUD, compass). Не для карточек в списках
3. **Accent glow** -- `box-shadow: 0 4px 28px accent 44%` -- только для кнопок Start Ride и Get Premium
4. **Hex** -- всегда pointy-top ориентация
5. **Карта** -- Stadia Outdoors dark theme, тёмный фон `#0B1118`
6. **Separator** -- `rgba(255,255,255,0.08)` между элементами в grouped-секциях
7. **MonoLabel** -- стандартный паттерн для подписей: mono 9px, 500, caps, tracking 0.12em, цвет `tq`
8. **MonoVal** -- стандартный паттерн для значений: mono 22px, 600, line-height 1
