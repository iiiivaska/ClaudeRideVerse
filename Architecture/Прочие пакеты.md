# Прочие пакеты
> WorkoutKit, TrackExport, PersistenceCore, DesignSystem. Вспомогательные переиспользуемые пакеты, не входящие в основные группы.

---

## WorkoutKit

**Назначение:** обёртка над HealthKit -- `HKWorkoutSession` wrapper, route insertion fix.

| Свойство | Значение |
|---|---|
| Зависимости | HealthKit |
| Публичный контракт | HKWorkoutSession wrapper, route insertion fix |

Инкапсулирует работу с HealthKit для записи тренировок. Включает workaround для критического gotcha FB18603581 (route insertion). Отвечает за:
- Запуск/остановку `HKWorkoutSession`.
- Вставку маршрута (`HKWorkoutRouteBuilder`).
- Сохранение тренировки в HealthKit.

[[LocationKit|LocationRecording]] не вставляет в `HKWorkoutRouteBuilder` -- это обязанность WorkoutKit.

| Свойство | Значение |

## PersistenceCore

**Назначение:** GRDB helpers, миграции, WAL-конфигурация, observation helpers.
**Статус:** реализован в Phase F ([[../Roadmap/Phase F — Foundation|F.5 — SCRUM-20]]), коммит `84f24fb`. Живёт в `Packages/PersistenceCore/`, публичные API в `import PersistenceCore`.

| Свойство | Значение |
|---|---|
| Зависимости | GRDB 7.10.0 (SPM) |
| Платформа | iOS 26 + macOS 26 (macOS для `swift test` на CI) |
| Публичный контракт | `DatabaseProvider`, `Migration` protocol, async observation, batch writes |
| README | `Packages/PersistenceCore/README.md` |

### Что внутри

- **`DatabaseProvider`** (`final class`, `Sendable`) — `DatabasePool` с WAL-конфигурацией:
  - `PRAGMA journal_mode = WAL` — конкурентные чтения во время записи
  - `PRAGMA synchronous = NORMAL` — баланс safety/speed
  - `PRAGMA journal_size_limit = 4194304` — лимит WAL 4 MB
  - `init(at:migrations:)` — file-based, `static func inMemory(migrations:)` — тесты/previews
  - `vacuum()` — периодическая очистка после bulk-удалений
- **`Migration` protocol** — статический протокол (`identifier` + `migrate(_ db:)`), обёртка над `DatabaseMigrator`
- **`observe(_:)`** — GRDB `ValueObservation` → `AsyncValueObservation<T>`
- **`writeInTransaction(_:)`** — batch-записи в транзакции (критично для 1M+ GPS-точек)
- **`@_exported import GRDB`** — downstream-потребители получают GRDB-типы без отдельного импорта

### Тесты

13 тестов Swift Testing: WAL mode, synchronous NORMAL, миграции (создание, порядок, идемпотентность), in-memory factory, vacuum, batch write 1000 rows, concurrent reads during write, observation.

Используется в [[Domain|FogRidePersistence]] для конкретных таблиц и репозиториев.

## DesignSystem

**Назначение:** FogRide design tokens -- цвета, типографика, общие компоненты.
**Статус:** реализован в Phase F ([[../Roadmap/Phase F — Foundation|F.3 — SCRUM-18]]), PR #2. Живёт в `Packages/DesignSystem/`, публичные API в `import DesignSystem`.

| Свойство | Значение |
|---|---|
| Зависимости | SwiftUI (и только SwiftUI -- см. [[Антипаттерны]]) |
| Платформа | iOS 26 + macOS 26 (последняя добавлена только чтобы `swift build` работал на Mac CLI) |
| Публичный контракт | tokens: colors, fonts, spacing/radii/shadow, glass modifier; 20+ SwiftUI компонентов |
| README | `Packages/DesignSystem/README.md` |

### Что внутри

| Группа | Публичные сущности |

| Tokens | `Color.fog*` (bg/surface1-3, accent+soft+border, status, text tiers, glass), `Font.fog*` (title/heading/body/button/caption/micro + monoXL/lg/md/sm/cap), `FogSpacing`, `FogRadius`, `FogShadow`, `.fogGlass(level:)` для `.tabBar`/`.hud`/`.control` |
| Primitives | `MonoLabel`, `MonoVal(size:color:)`, `GlassPill`, `GlassCircle` |
| Buttons | `.buttonStyle(.fogPrimary)`, `.fogSecondary`, `.fogLink` + `StopButton` |

| Cards | `GlassHUDCard`, `MetricCell4Grid`, `StatCard2x2`, `GroupedCard`, `UpsellCard` |

| Gallery | `DesignSystemGallery` (публичный embeddable view) + Xcode app target `DesignSystemGallery` для Simulator |

### Ключевые решения по реализации

- **Шрифты: SF Pro + SF Mono**, не DM Sans + IBM Plex Mono. Custom-font registrar убран — упрощает пакет, убирает bundled `.ttf`, не требует `resources: [.process(...)]`. Переключение обратно на кастомные шрифты — локальное изменение в `Font+FogRide.swift`, call-sites не меняются.
- **GlassStyles = ViewModifier**, не только Color-константы. Тройка `.tabBar` / `.hud` / `.control` ViewModifier'ов — чтобы `GlassPill` / `GlassHUDCard` / `FloatingGlassTabBar` не переизобретали свой glass.
- **PulseAnimation = один общий modifier**. `RecPulseDot` / `LocationPulseRing` / hex glow — тонкие обёртки.
- **Тесты:** 49 Swift Testing assertions в 13 suites — пока `ImageRenderer` smoke-checks. Настоящие screenshot baselines отложены в [[../Roadmap/Phase F — Foundation#Follow-up|SCRUM-72]].

Если `TripDetailFeature` нужно поделиться кнопкой с `RecordingFeature`, кнопка выносится сюда (см. [[Антипаттерны]]).
